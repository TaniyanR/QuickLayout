# QuickLayout

HTML / CSS / JavaScript を入力し、**「反映」ボタンを押した時だけ** PC とスマホのレイアウトを確認できる CodePen 風の軽量ツールです。外出先（iPhone）での利用・配布を前提に、静的ファイルのみで動作します。

## QuickLayout の概要
- HTML / CSS / JavaScript の 3 分割エディタ
- プレビューは [PC] / [スマホ] タブで切り替え
- 反映ボタン式（自動反映なし）
- 下書き自動復元（localStorage）
- 端末内に最大 10 件まで保存（localStorage）
- スクショ（PNG）を端末に保存

---

## 「反映」ボタンの使い方
1. HTML / CSS / JavaScript を入力
2. **「反映」** を押す
3. [PC] / [スマホ] タブを切り替えて確認

※ 入力中は自動でプレビュー更新されません。

---

## 下書き自動復元
- 編集中の内容は 1〜2 秒のデバウンスで localStorage に自動保存されます。
- リロード・再訪問時に自動復元されます。
- 復元後も **プレビューは自動更新されません**。必ず「反映」を押してください。

---

## 10件保存（端末内）
- 保存名を入力して **保存** を押すと localStorage に記録されます（最大 10 件）。
- 保存名未入力の場合は日時で自動命名されます。
- 同名の場合は上書き確認が表示されます。
- 10 件満杯の場合は「上書き or 削除」を案内します。
- 読み込み後もプレビューは自動更新されません。必要なら「反映」を押してください。

---

## スクショの保存方法（iPhone 手順）
1. [PC] / [スマホ] タブで表示したい方を選ぶ
2. **「反映」** を押す
3. **「スクショ」** を押す
4. ダウンロードされた PNG を「ファイル」アプリで確認

- PC タブ: `quicklayout_pc_YYYY-MM-DD_HHMMSS.png`
- スマホタブ: `quicklayout_sp_YYYY-MM-DD_HHMMSS.png`
- 解像度は pixelRatio=2（高解像度）

---

## 設置方法（アップロードのみ）
以下のファイルをそのままサーバーにアップロードしてください。

```
/index.html
/assets/app.css
/assets/app.js
/README.md
```

---

## 注意事項
- 保存は localStorage のみ（端末依存）です。
- ブラウザデータ削除で保存内容は消えます。
- 入力した JavaScript はそのまま実行されます。
- 外部画像 / 外部 CSS があるとスクショが失敗する場合があります。
