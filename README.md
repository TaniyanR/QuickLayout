# QuickLayout

HTML / CSS / JavaScript を入力して、**PCレイアウト**と**スマホレイアウト**をすぐ確認できる  
CodePen風の軽量プレビューアです。

- ✅ **反映ボタン式**（押した時だけプレビュー更新）
- ✅ **下書きの自動復元**（localStorage）
- ✅ **端末内に最大10個まで保存**（localStorage / サーバー保存なし）
- ✅ **PC / スマホ** どちらの画面でも使いやすいUI

---

## できること

### 1) レイアウト確認（PC / スマホ）
- 上段（PC）は **HTML / CSS / JavaScript** を3分割で編集
- 下段プレビューはタブで **[PC] / [スマホ]** を切り替え
- スマホタブは **幅375px相当** のプレビューで表示（端末枠風）

### 2) 反映（プレビュー更新）
- 入力したコードは **「反映」ボタンを押した時だけ** プレビューに適用されます  
  ※入力中は勝手にプレビューが変わらないので、挙動が安定します。

### 3) 下書き自動復元（必須）
- 編集中の内容は自動的に端末内へ下書き保存されます（localStorage）
- リロードや再訪問でも、前回の下書きを復元できます  
  ※ただしプレビュー更新は **必ず「反映」** を押してください。

### 4) 10個まで保存（端末内）
- “作業スナップショット”として、端末内（localStorage）に **最大10個** 保存できます
- 「保存」「読み込み」「削除」をサポート
- 10件満杯の場合：
  - 既存データの **上書き** または
  - **削除して空ける**
  を選んでください

> ⚠️ 注意：保存は端末内のみです。別端末へは自動で引き継がれません。

---

## 使い方

### 反映
1. HTML / CSS / JavaScript を入力
2. **「反映」** を押す
3. 下段プレビューで **[PC] / [スマホ]** を切り替えて確認

### 保存（最大10件）
1. 保存名を入力（未入力なら日時で自動命名）
2. **「保存」** を押す
3. 一覧に追加されます（最大10件）

### 読み込み
1. 一覧から保存データを選ぶ
2. **「読み込み」** を押す  
   → エディタに反映されます  
   ※プレビューは自動更新されないので、必要なら **「反映」** を押してください。

### 削除
- 一覧から対象を選び **「削除」**
